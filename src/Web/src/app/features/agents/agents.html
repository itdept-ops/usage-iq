Proactive Agents

Per-you assistants that run on a daily schedule and nudge you — a morning briefing, a late-day streak rescue, a budget alert, and a low-staples reminder. Each lands in your in-app bell (and on your devices if you've enabled web-push). Agents only ever watch your own data and only ever nudge you; AI summaries use your existing AI access — the deterministic version is always there without it.
